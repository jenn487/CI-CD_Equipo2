## Proyecto TDD - Flujo Básico de CI/CD en GitHub

Este repositorio contiene el proyecto de Test-Driven Development (TDD) acompañado de la configuración de un flujo básico de Integración Continua (CI) y Despliegue Continuo (CD) utilizando GitHub Actions.
El objetivo es automatizar la compilación y validación del código, asegurando calidad y eficiencia en el desarrollo colaborativo.
